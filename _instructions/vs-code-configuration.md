 * Enable Auto Save
 * Set Color Theme to "Solarized Dark"
 * Set File Icon Theme to "Seti"

Or something link this:

````json
{
    "files.autoSave": "onFocusChange",
    "workbench.colorTheme": "Solarized Dark",
    "workbench.iconTheme": "vs-seti"
}
````
